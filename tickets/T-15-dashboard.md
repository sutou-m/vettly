# T-15 ダッシュボード（候補者数・スコア分布・ポジション別集計）

## 担当エージェント
@frontend-developer @ui-designer

## 目的
採用活動の全体状況を一目で把握できるダッシュボードを実装する。
候補者視点・ポジション視点の両方を提供する。

## 前提チケット
- T-07・T-08完了済みであること

## 完了条件
- [ ] `/dashboard` にサマリカードが表示される
- [ ] ステータス別候補者数の棒グラフが表示される
- [ ] ポジション別候補者一覧・平均スコアが表示される
- [ ] データ0件のEmpty Stateが表示される

## 表示内容

### サマリカード（4枚）
| カード | 内容 |
|--------|------|
| 総候補者数 | 全ポジション合計 |
| 今週の新規 | 直近7日間のアップロード数 |
| 平均スコア | 全候補者の平均AIスコア |
| 面接調整中 | statusが'interview'の件数 |

### グラフ
- ステータス別候補者数：横棒グラフ（recharts）
- スコア分布：ヒストグラム（0-49/50-74/75-100の3段階）

### ポジション別サマリテーブル
| ポジション | 応募者数 | 平均スコア | 最高スコア候補者 | 未処理 |
|-----------|---------|-----------|----------------|-------|

```bash
npm install recharts
```

## 注意事項
- rechartsはClient Component内でのみ使用（`'use client'`必須）
- グラフコンポーネントだけ切り出してClient Componentにする
