# T-08 候補者一覧（カンバンビュー・ステータス更新）

## 担当エージェント
@frontend-developer @ui-designer

## 目的
候補者をステータス別カンバンビューで管理する。
Vettlyのメイン画面であり、ポートフォリオとして最も映える画面。

## 前提チケット
- T-03・T-04・T-05・T-07完了済みであること

## 完了条件
- [ ] `/candidates` にカンバンビューが表示される
- [ ] ステータス別カラム（6列）に候補者カードが表示される
- [ ] ステータス更新ボタンで候補者を移動できる
- [ ] 候補者カードにスコアバー・スキルタグが表示される
- [ ] ポジションでフィルタできる
- [ ] データ0件のEmpty Stateが表示される

## カンバム構成

```
| 書類選考 | レビュー中 | 面接中 | 内定 | 不採用 | 辞退 |
|---------|----------|-------|------|-------|------|
| card    | card     | card  |      |       |      |
| card    |          |       |      |       |      |
```

### 候補者カードの表示内容
- 氏名（AI抽出）
- ポジション名
- AIスコア（ScoreBarコンポーネント）
- スキルタグ（上位3件）
- 処理日時
- 「詳細を見る」リンク

### ステータス更新
- カード内の「→ 次のステージへ」ボタン
- 「不採用」「辞退」ボタン（確認ダイアログあり）

### フィルタ
- ポジション別フィルタ（selectボックス）
- スコア範囲フィルタ（高スコア順ソート）

## テーブル
- `vet_candidates`・`vet_positions`（vet_プレフィックス必須）

## 注意事項
- カンバンは横スクロール対応（6列あるため）
- Server ComponentでデータFetch + Client Componentでインタラクション
